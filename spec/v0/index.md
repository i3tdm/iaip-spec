# IAIP Specification v0

The Instructions AI Protocol (IAIP) defines a deterministic, repository-based
orientation system for AI/LLM Consumers.

This specification describes:
- how instruction files are structured
- how they are discovered
- how they are resolved
- how Consumers must behave

This document set is normative for IAIP v0.

## Status

- Version: v0
- Stability: Draft (breaking changes allowed)
- Scope: Orientation only (no execution)

## Normative Language

The key words **MUST**, **MUST NOT**, **SHOULD**, **SHOULD NOT**, and **MAY**
are to be interpreted as described in RFC 2119.
