# Status e Filosofia

### Estado Atual
O IAIP é considerado **experimental**, mas já é utilizado diariamente em fluxos reais. É um "draft utilizável" onde a base de orientação já é sólida o suficiente para consumo.

### Filosofia: Determinismo > Inteligência
Em IA, se você não sabe para onde ir, qualquer lugar serve. O IAIP é determinístico para evitar que resumos de contexto deixem "resquícios" que viram regras erradas para a IA. 

Orientar é passar dados necessários para minimizar alucinações. Diferente da automação (que executa fluxos), a orientação define as regras que a automação deve respeitar.
